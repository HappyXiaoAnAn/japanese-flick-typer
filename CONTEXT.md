# Domain Glossary

## 九宮格滑動鍵盤 (9-Key Flick Keyboard)
網頁畫面上模擬的 3x4 日文打字鍵盤。使用者不透過行動裝置的原生鍵盤輸入，而是直接在畫面的虛擬按鍵上進行觸控（或滑鼠）點擊與滑動。

## 滑動輸入 (Flick Input / フリック入力)
日文九宮格鍵盤的輸入機制。點擊按鍵中心輸入預設的「あ段」母音字元（如「あ」）；按住並朝左、上、右、下四個方向滑動，則分別輸入「い段」、「う段」、「え段」、「お段」的字元。

## 修飾鍵 (Modifier Key / 小 ﾞ ﾟ)
模擬鍵盤左下角的按鍵。用於變更前一個輸入的假名，使其加上濁點（如「か」→「が」）、半濁點（如「は」→「ぱ」）或轉換為小寫字（如「つ」→「っ」、「や」→「ゃ」）。

## 提示機制 (Hint Mechanism)
在練習過程中，系統提供的輔助視覺提示。啟用時，網頁上的虛擬鍵盤會高亮下一個目標字元所在的按鍵，並以箭頭指出滑動方向，協助使用者記憶與熟悉九宮格的字元分佈。

## 五十音單音練習 (Kana Row Practice)
基礎練習模式。使用者可自由選擇特定的五十音「行」（如あ行、か行），系統會隨機出題該行之假名以熟悉鍵盤按鍵對應。

## 內建精選詞庫 (Built-in Vocabulary)
預設於程式中的日文單字與短句資料庫。每一筆資料皆預先定義了「顯示用漢字」與對應的「目標輸入假名」，避免瀏覽器端進行繁重之漢字轉假名運算。

## 自訂詞庫 (Custom Vocabulary)
進階練習模式。使用者可以自由輸入或貼上純假名/平假名與片假名文字，系統會依此產生打字練習題目。

## 字符速度 (Characters Per Minute / CPM)
日文打字的輸入速度指標。計算公式為：(輸入正確假名數 / 總輸入時間以分鐘計)。在日文中比單字速度（WPM）更具參考價值。

## 弱點分析與鍵盤熱力圖 (Weak Key Analysis & Keyboard Heatmap)
系統對使用者輸入錯誤進行追蹤的機制。系統會記錄各個按鍵與滑動方向的錯誤頻率，並在虛擬鍵盤上以顏色深淺（熱力圖）標示，幫助使用者找出最容易按錯的「行」或「滑動方向」。
